frc-scout-2013

Web-based scouting application for the 2013 FRC game. The FIRST Robotics Competition is a worldwide event in which teams build and compete with robots. FRC, the high school level, is the most difficult, with the largest robots, shortest build seasons, and most complex tasks. You can read more about it here.

This application aims to make it easier for teams to scout other teams in matches, to make selections for the finals easier provided they both make it. You can try the 2012 version out over at frcscout.com.

If you'd like to take a look at the code, you can head over to the repository or clone it:

git clone https://github.com/terabyte128/first-scout-2013.git

Happy scouting! :)
